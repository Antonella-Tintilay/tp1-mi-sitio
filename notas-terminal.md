# Comandos usados
## 1.pwd
**Funcion:** Mostrar la carpeta en donde se encuentra en ese momento
**situacion:** Asegurarme de estar dentro de la carpeta de mi proyecto antes de crear los archivos
## 2.ls
**Funcion:** listar los archivos y carpetas del directorio actual
**situacion:** Ver si se creo correctamente mi archivo en la carpeta
## 3.cd
**funcion:** Permite entrar a la carpeta, cambio de directorio
**situacion:** Entar a mi carpeta para interactuar con mis archivos
## 4.mkdir:
**funcion:** Crea una nueva Carpeta
**situacion:** Creé mi carpeta tp1-mi-sitio
## 5.touch:
**funcion:** Crear un nuevo archivo vacio
**situacion:** Creé index.html
## 6.cat:
**funcion:** Muestra el contenido de un archivo
**situacion:** Revise el contenido de index.html
## 7.whitch:
**funcion:** Muestra la ruta de programa o comando
**situacion:** Ver en donde se instaló "git"
## 8.code: 
**funcion:** Abre VS Code desde la terminal
**situacion:** Abrí la carpeta del proyecto directamente con "code" en la terminal
## 9.rm: 
**funcion:** Elimina una archivo o carpeta
**situacion:** Borre archivos por el directorio incorrecto cuando intentaba crear la carpeta de mi proyecto
## 10.mv:
**funcion:** Mueva o renombra archivos
**situacion:** Movi imagenes descargadas a otra carpeta
